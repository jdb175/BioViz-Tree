Tree of Life


   This is radial dendrograph of hierarchically clustered data from the Zoo dataset. Clicking or hovering on leaf nodes will show information about them in the table below. Hovering on a branch will show all shared characteristics among the children of that branch in the table. Clicking on a branch node will filter the tree to show the subtree beginning with that branch. When viewing a subtree, you can return to the root by clicking the small tree in the top left.
